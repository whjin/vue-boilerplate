# Vue项目模板目录结构 #

![](https://raw.githubusercontent.com/whjin/images-save/master/vue-boilerplate/vue-boilerplate.jpg)

|目录|说明|
|----|----|
|`build`|项目构建（`webpack`）相关代码|
|`config`|构建配置目录|
|`API-Schema`|页面接口`API`目录|
|`mock`|存放模拟数据|
|`src`|源码目录|
|`assets`|资源目录|
|`components`|组件目录|
|`common`|共用方法|
|`http`|请求方法封装|
|`pages`|项目页面目录|
|`store`|共用数据仓库|
|`router`|路由配置目录|
|`App.vue`|页面级`Vue`组件|
|`main.js`|页面入口`JS`文件|
|`static`|静态文件目录，比如一些图片，`json`数据|
|`index.html`|入口文件|
|`.editorconfig`|`ES`语法检查|
|`.babelrc`|`ES6`语法编译配置|
